## 미결 사항 종합

개발 전 또는 개발 중 확정 필요:

- [ ] 시료 전처리 매트릭스 기본값 (질산 희석 vs 유기용매 직접 주입) — 배경 원소 기본 토글 결정
- [ ] 간섭 심각도 등급 경계값 (존재비 곱 기준 %)
- [ ] 즐겨찾기 화합물 전체 목록과 정확한 분자식 (3DMAS, BTBAS, OMCTS, CPME 등)
- [ ] 보유 장비의 DRC 셀 가스 종류 (NH3/O2/H2/He 중 실제 사용 가능한 것)
- [ ] 이중하전(M²⁺) 간섭 적용 범위 (Ba·REE 등 한정 vs 전 원소)
- [ ] 사내망 서버 PC / AWS 보안 정책 (Phase 3 시점)
