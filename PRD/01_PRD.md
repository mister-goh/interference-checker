## 1. 제품 개요

### 한 줄 요약
ICP-MS 분석 전 "어떤 질량(동위원소)으로 측정해야 간섭 없이 정확한가"를 메인 물질 입력 한 번으로 전 원소에 대해 자동 추천하는 웹앱.

### 해결하는 문제
- 반도체 소재(Metal Organic, Si계 Organic, Metal Halide, 기타 유기물)의 초극미량 Metal 분석 시, 메인 물질에 과량 포함된 원소가 동질량 간섭·다원자 이온 간섭(예: 플라즈마에서 생성되는 ⁴⁰Ar³⁵Cl⁺ → ⁷⁵As 방해)을 일으킴
- 현재는 분석자가 문헌 간섭표를 수동으로 뒤져가며 질량을 선정 → 시간 소모 + 누락 위험
- 보증해야 하는 Metal 수가 계속 늘어나 수동 검토의 한계 도달

### 핵심 가치
- 기존 대안(USGS/Thermo 등의 정적 PDF 간섭표)은 "조회"만 가능 — 이 앱은 **메인 물질을 입력하면 매트릭스 조합 간섭까지 자동 계산**해 원소별 최적 질량을 제안
- 쿼드러폴(단위분해능 1 amu) 장비 현실을 반영: 정수 질량이 겹치면 "분석 불가"로 판정
- 피할 수 없는 간섭은 DRC/KED 모드 + 셀 가스까지 함께 추천

## 2. 사용자

### 주요 사용자
- **누구**: 품질관리팀 QC Part 분석원 (ICP-MS 운용자), 추후 사내 전체 구성원
- **상황**: 신규 물질 분석법 개발 시, 보증 원소 추가 시, 간섭 의심 결과 검토 시
- **목표**: 간섭 없는 분석 질량을 빠르고 빠짐없이 선정하고, 근거를 보고서/SOP에 남김

### 사용자 시나리오
1. 분석원이 신규 물질(예: HfCl4) 분석법 개발을 시작하며 앱을 열고
2. 즐겨찾기에서 HfCl4를 클릭(또는 화학식 직접 입력)하면
3. 전체 원소에 대해 "추천 질량 + 간섭 목록 + 권장 모드(Standard/KED/DRC)" 테이블이 즉시 출력되고, 질량/원소기호 순으로 정렬해 확인 후 Excel로 내보내 SOP에 첨부한다

## 3. 핵심 기능

| 기능 | 설명 | 우선순위 | 복잡도 |
|------|------|----------|--------|
| 동위원소 데이터베이스 내장 | 전 원소 동위원소 질량·자연존재비(NIST 기준) 정적 데이터로 내장 | P1 (MVP) | 간단 |
| 화학식 파서 + 즐겨찾기 | HfCl4 등 화학식 자동 해석 + 자주 쓰는 화합물(3DMAS, BTBAS, OMCTS, CPME 등) 클릭 선택 | P1 (MVP) | 보통 |
| 간섭 계산 엔진 | 동질량 + Ar 다원자 + 매트릭스 조합(산화물 MO⁺, 수산화물 MOH⁺, 수소화물 MH⁺, 염화물 MCl⁺, 아르곤화물 ArM⁺, 이중하전 M²⁺) 자동 생성·판정 | P1 (MVP) | 복잡 |
| 쿼드러폴 단위분해능 판정 | 정수 질량(1 amu) 겹침 = 간섭으로 판정, 분리 불가 표시 | P1 (MVP) | 간단 |
| 추천 질량 결과 테이블 | 원소별 최적 동위원소 추천, 질량·원소기호 오름/내림차순 정렬 | P1 (MVP) | 보통 |
| DRC/KED 모드 추천 | 간섭 회피 불가 시 He KED / NH3·O2·H2 DRC + 셀 가스 추천 | P1 (풀 패키지) | 보통 |
| 간섭 근거 상세보기 | 질량 클릭 → 간섭종 조성·존재비·심각도 표시 (신입 교육용 겸용) | P1 (풀 패키지) | 보통 |
| CSV/Excel 내보내기 | 결과 테이블을 보고서·SOP용으로 다운로드 | P1 (풀 패키지) | 간단 |
| 간섭 없는 원소만 필터 | Interference-free 질량 보유 원소만 보기 | P2 | 간단 |
| 사내망 공유/AWS 배포 | 팀 공유 → 추후 AWS 정적 호스팅 이전 | P3 | 보통 |
| 주기율표 시각화 뷰 | 원소별 간섭 상태를 주기율표 색상으로 표시 | P3 (선택) | 복잡 |

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름
```
앱 열기 -> 메인 물질 입력(화학식 타이핑 or 즐겨찾기 클릭)
        -> 배경 매트릭스 확인(Ar 기본 + H/N/O/C 토글)
        -> [계산] -> 원소별 추천 질량 테이블
        -> 정렬/상세보기 -> Excel 내보내기
```

### 상세 흐름
1. **물질 입력**: 화학식 입력창에 `HfCl4` 타이핑 → 자동 파싱되어 구성 원소(Hf, Cl) 표시. 또는 즐겨찾기 칩(3DMAS, BTBAS, OMCTS, HfCl4, YCl3, SnCl4, CPME 등)을 클릭
2. **배경 매트릭스 설정**: Ar(플라즈마)은 항상 포함. 물/질산 유래 H, N, O와 유기물 유래 C는 기본 ON, 토글로 조정 가능
3. **계산**: 매트릭스 원소들의 동위원소로 만들 수 있는 모든 다원자 조합(2원자 중심) + 동질량 + 이중하전을 정수 질량으로 매핑
4. **결과 확인**: 원소별로 ① 추천 질량 ② 간섭 여부 배지(없음/회피가능/모드필요/분석곤란) ③ 권장 모드 표시. 질량/원소기호 정렬 토글
5. **상세보기**: 행 클릭 → 해당 원소의 전체 동위원소와 각 질량의 간섭종 목록(조성, 존재비, 심각도) 패널 표시
6. **내보내기**: 현재 정렬·필터 상태 그대로 CSV/Excel 다운로드

## 5. 성공 기준

- [ ] HfCl4 입력 시 ⁴⁰Ar³⁵Cl⁺ → ⁷⁵As 간섭이 자동 검출되고 As에 대해 KED/DRC 모드가 추천된다
- [ ] 즐겨찾기 화합물(3DMAS, BTBAS, OMCTS, HfCl4, YCl3, SnCl4, CPME) 클릭 → 3초 이내 전 원소 결과 출력
- [ ] 결과 테이블이 질량/원소기호 각각 오름·내림차순 정렬되고, 정렬 상태 그대로 Excel로 내보내진다
- [ ] 추천 질량이 문헌 간섭표(USGS polyatomic interference table)와 대표 사례 10건 이상에서 일치한다
- [ ] 팀원이 브라우저로 접속해 설치 없이 바로 사용할 수 있다

## 6. 안 만드는 것 (Out of Scope)

> 이 목록에 있는 건 Phase 1에서 만들지 않습니다.
> AI에게 코드를 시킬 때 이 목록을 함께 공유하세요.

- **로그인/계정 기능** -- 이유: 개인 데이터 없음. 즐겨찾기는 브라우저 localStorage로 충분. AWS 배포 시점에 사내 SSO 검토
- **서버/백엔드/DB** -- 이유: 동위원소·간섭 데이터는 정적이므로 앱에 내장. 정적 호스팅만으로 동작해야 사내망→AWS 이전이 쉬움
- **정량 보정 계산(간섭 보정 방정식)** -- 이유: 이 앱은 "질량 선정" 도구. 보정 계수 계산은 장비 소프트웨어 영역
- **고분해능(HR-ICP-MS)·트리플쿼드(MS/MS) 모드** -- 이유: 보유 장비가 일반 쿼드러폴. 단위분해능 가정으로 단순화
- **3원자 이상 다원자 조합 전수 생성** -- 이유: 조합 폭발. 문헌에 보고된 주요 3원자 종(Ar2⁺, ArO H⁺ 등)은 큐레이션 목록으로 내장
- **모바일 전용 앱** -- 이유: 분석실 PC 사용이 주. 반응형 웹으로 충분
