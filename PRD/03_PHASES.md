## Phase 1: MVP — 핵심 계산 + 풀 패키지 결과 화면 (1~2주)

### 목표
메인 물질을 입력하면 전 원소 추천 질량 + 간섭 근거 + DRC/KED 추천까지 한 화면에서 확인하고 Excel로 내보낼 수 있다. (인터뷰에서 "풀 패키지" 선택을 반영해 핵심 기능을 모두 Phase 1에 포함)

### 기능
- [ ] 동위원소 데이터베이스 내장 (전 원소, NIST 존재비, 정적 TS/JSON)
- [ ] 화학식 파서 (HfCl4, SiH(N(CH3)2)3 등 괄호 포함 해석) + 즐겨찾기 기본 목록 (3DMAS, BTBAS, OMCTS, HfCl4, YCl3, SnCl4, CPME 등)
- [ ] 배경 매트릭스 토글 (Ar 고정 + H/N/O/C 기본 ON)
- [ ] 간섭 계산 엔진: 동질량 + Ar 다원자 + 매트릭스 2원자 조합(MO⁺/MOH⁺/MH⁺/MCl⁺/ArM⁺) + 이중하전 + 문헌 큐레이션 간섭 목록
- [ ] 단위분해능(1 amu) 기준 쿼드러폴 간섭 판정 + 심각도 등급
- [ ] DRC/KED 모드 + 셀 가스 추천 로직 (간섭 유형별 규칙 기반)
- [ ] 원소별 추천 질량 결과 테이블 (질량/원소기호 오름·내림차순 정렬)
- [ ] 간섭 근거 상세보기 패널 (행 클릭 → 간섭종·존재비·심각도)
- [ ] CSV/Excel 내보내기 (SheetJS)

### 데이터
- Element, Isotope, Compound(기본 내장), Interference, Recommendation — 전부 사용

### 인증
- 없음 (정적 앱, 개인 데이터 없음)

### "진짜 제품" 체크리스트
- [ ] 실제 NIST 동위원소 존재비 데이터 사용 (대충 만든 가짜 숫자 X)
- [ ] 대표 검증 케이스 통과: HfCl4 → ⁴⁰Ar³⁵Cl⁺/⁷⁵As 검출, ⁴⁰Ar¹⁶O⁺/⁵⁶Fe 검출, Ar2⁺/⁸⁰Se 검출
- [ ] 계산 엔진 단위 테스트 작성 (문헌 간섭표 대조 10건 이상)
- [ ] `npm run build` 결과물(dist)을 브라우저로 열어 실제 동작 확인 (localhost 개발모드만 X)
- [ ] 팀원 1명이 dist 폴더를 받아 본인 PC에서 사용해봄

- DRC/KED 모드 추천

## Phase 2: 확장 — 실무 활용도 (1주)

### 전제 조건
- Phase 1 검증 케이스 통과 + 팀원 1명 이상 실사용 피드백 수집

### 목표
일상 분석 업무에서 "쓰기 편한" 도구로 다듬는다.

### 기능
- [ ] 간섭 없는 원소만 보기 필터 + 판정 배지별 필터
- [ ] 보증 원소 목록 프리셋 (자주 보는 원소 세트 저장)
- [ ] 즐겨찾기 사용자 커스텀 추가/수정/삭제 (localStorage)
- [ ] 심각도 기준치 설정 화면 (실측 대비 조정 가능하게)
- [ ] 인쇄 친화 레이아웃 (SOP 첨부용)

### 추가 데이터
- 사용자 설정(UserSettings: 기준치, 프리셋) — localStorage 저장

### 통합 테스트
- Phase 1 검증 케이스 10건이 여전히 통과하는지 확인

## Phase 3: 고도화 — 공유·배포 (1~2주, AWS 도입 시점 연동)

### 전제 조건
- Phase 1 + 2가 팀 내에서 안정적으로 사용 중

### 목표
사내망 공유 → AWS 전사 배포, 시각화 고도화.

### 기능
- [ ] 사내망 공유: 사내 PC 1대에서 정적 서버로 서빙 (팀원 URL 접속)
- [ ] AWS 배포: S3 + CloudFront 정적 호스팅 (빌드 결과물 그대로 업로드)
- [ ] 주기율표 시각화 뷰 (원소별 판정 상태 색상 표시, 선택)
- [ ] 사내 접근 통제 필요 시 SSO 검토 (IT 협의)

### 주의사항
- AWS 비용: 정적 호스팅이라 월 수 달러 수준이나 사내 계정 정책 확인 필요
- 사내 분석 물질 정보가 외부 인프라에 올라가는 것에 대한 보안 검토 선행
